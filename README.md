# PRS Risk Interpreter

Interactive computational genomics platform visualizing ancestry-dependent polygenic risk score (PRS) distortion and threshold recalibration using 1000 Genomes data.

## Current Features
- EUR vs SAS PRS distributions
- Threshold distortion visualization
- Percentile recalibration
- Genomic risk interpretation

## Research Motivation
Polygenic risk scores are increasingly used in precision medicine, but PRS interpretation can vary substantially across populations due to allele-frequency differences, linkage disequilibrium structure, and training bias. This platform aims to visualize and explain these effects interactively.
